# Sentei

Git branch cleaner built with Node.js

## Installation

Run once

```bash
$ npx sentei
```

Global install

```bash
$ npm i -g sentei
$ sentei
```

## Usage

```bash
  Usage
    $ cd /path/to/<your-project>
    $ sentei

  Options
    --force, -f force delete
    --safety, -s protect branch if ['master', 'develop', 'release'].includes(branch)
```

## Contribution

```bash
# Install packages
$ yarn

# Build code
$ yarn build

# Test
$ yarn test

# Run code
$ node ./bin/cli.js
```

and send Pull request!
