Git branch cleaner built with Node.js

## Installation

Run once

```bash
$ npx sentei
```

Global install

```bash
$ npm i -g sentei
$ sentei
```

## Usage

```bash
  Usage
    $ cd /path/to/<your-project>
    $ sentei

  Options
    --force, -f force delete
    --all, -a include master, develop, release branches for deletion
```
